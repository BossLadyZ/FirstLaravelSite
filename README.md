## My First Laravel Project

I built a mini social media post system using Laravel as a way to get acquainted with Laravel and PHP. The app has a user authentication 'system' that stops guests from carrying out acts like "Post", "Liking/Unlike" a post. It also has a register and a login form; both of which work well. I also learnt
how to read data from a database using PHPMyAdmin
